How to Install APK + OBB File on Android

1. Make sure that third-party apps are allowed on your device: go to Menu->Settings->Security->and check “Unknown Sources”
2. Download APK and OBB file and open the APK file, tap apk file, then hit Install and do not run yet.
3. After successful installation, you need to place OBB file in the place /Android/obb/com.pubg.imobile/main.15336.com.pubg.imobile.obb
